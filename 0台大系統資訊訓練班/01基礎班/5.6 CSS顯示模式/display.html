顯示模式

區塊元素(block)代表: h1~h6,div,section,header,p,br,li,ul,ol...

行內文字元素(inline)代表: a, span, label

行內區塊元素(inline-block)代表: input, img, button

div
a

區塊元素(block)

                
選擇目標 {
    /* 設定此元素為區塊元素 */
    display: block;
}
  • 預設:h1~h6,p,ul,ol,li,div,header,section,footer,nav,....
  • 會造成換行
  • 可設定width與height
div
a

行內文字元素(inline)

                
選擇目標 {
    /* 設定此元素為行內文字元素 */
    display: inline;
}
  • 預設:a,span,label,:before,:after
  • 不會造成換行
  • 無法設定width,height
  • 中間會有空格
div
anchor

行內區塊元素(inline-block)

                
選擇目標 {
    /* 設定此元素為行內區塊元素 */
    display: inline-block;
}
  • 預設:img,button,input,textarea,select
  • 不會造成換行
  • 可以設定width,height
  • 中間也會有空格
